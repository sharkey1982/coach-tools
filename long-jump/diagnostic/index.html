Long Jump / Diagnostic

Enter a child's 60m time, CMJ, height, and best actual jump. The tool calculates physical potential, current efficiency, and which technique factor has the biggest coachable gain. Save each athlete for whole-group analysis.

1 Input

Measurements

Technique scoring — score each by eye, 1 (poor) to 5 (excellent)
How the formula works Speed comes from 60m time using v = 60 / (t − 1). Board speed is 88% of top speed (short run-up). Vertical velocity comes from CMJ via v_v = √(2g·h) with an 87% one-foot conversion. Predicted air distance is (2 × v_h × v_v) / g. Body geometry adds ~0.5 × height. Technique scores adjust the conversion of this physical potential into actual measured distance.
